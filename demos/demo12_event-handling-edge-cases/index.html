Case Two: .addEventListener as a SubFunction

Say for example when you buy an item from Amazon, you need to execute a lot of sub functions at the same time.

This process allows you to execute various functions a la misma vez.

(console.log's the original call location of the function aswell (this button).)

Format

item.addEventListener("click", () => { //Subfunction Code Here });

Caveats

  • If parsing an event, make sure your subfunction lists it and each subfunction uses it.
    • item.addEventListener("click", event => { newMethod(event); });
