JSON Basic Syntax

  • [] = Collection
  • {} = Object
  • "key": value = format
    • let person = { "firstName": "Kendra", "lastName": "Kindrai", "email": "kkindrai@madisoncollege.edu", "age": 23 };
  • , = seperator character

How to Access JSON Data

Dot Notation

Performing console.log(`firstName: ${person.firstName} \n lastName: ${person.lastName}`) will return the following:

  • firstName: Kendra
  • lastName: Kindrai

Collections of Objects [{}, {}, {}]
